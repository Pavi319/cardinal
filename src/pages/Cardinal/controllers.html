This page is intending to display how Controllers are written in this framework and how it is combined with Rivets.Js, a framework used for data-binding.

In this page, we will display 3 examples of templates, all of them being displayed using the same defined model.

On the top of the container, stands a button which is responsable for manipulating the model. So, everytime when that button is clicked, a new card is added in the page.

A great feature from Rivets is that you can customize the view as you want, so for all these examples, some custom binders and formatter have been implemented.
