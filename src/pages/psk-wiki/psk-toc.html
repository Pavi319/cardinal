This component creates a Table of Content in which every item is a link to that item's position in our psk-page. It renders a psk-card with the given title in which every psk-page and psk-chapter is rendered.

This components selects all psk-chapter components and creates a ordered list based on each chapter guid.

It renders a unordered list with it's HTML children.

This component has two child components:

  • First is the index of the element (table of content item index).
  • Second is the title of each chapter that is rendered on that psk-page.
