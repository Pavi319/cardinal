This component will render either a bullet list(using the same special characters as in Office) or an numeric list, based on the values given to the properties .

Every element in this list will be delimited by a new line .

The info sent between the tags will be processed in this way:

First of all every element delimited by new line with a length bigger than 0 will be passed in a array.

After this for every array element we verify if it's either a inline,start or end tag.

If it is any of the above we check for children,in a positive case we add them to a final array that will be rendered, and in a negative case we add only the line to the final array.
